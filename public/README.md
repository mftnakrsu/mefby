# Public Folder

Bu klasöre favicon (tarayıcı sekmesi ikonu) dosyanızı ekleyin.

## Favicon Ekleme

1. Fotoğrafınızı bu klasöre kopyalayın
2. Dosya adını `favicon.png` olarak değiştirin (veya mevcut adı koruyun)
3. Önerilen boyutlar:
   - 32x32 px (favicon)
   - 180x180 px (Apple touch icon)
   - 512x512 px (genel kullanım)

## Desteklenen Formatlar

- PNG (önerilen)
- ICO
- SVG

Not: index.html'de favicon.png olarak ayarlanmıştır. Farklı bir dosya adı kullanırsanız index.html'i güncelleyin.
